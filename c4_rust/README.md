# c4_rust_rashedd

## Build and Run Instructions

1. Install Rust from https://www.rust-lang.org/ (if not already installed).
2. Clone the repository:
   ```bash
   git clone https://github.com/R4SHHED/c4_rust_rashedd.git
   cd c4_rust_rashedd
   ```
3. Build the project:
   ```bash
   cargo build
   ```
4. Run the project:
   ```bash
   cargo run
   ```
